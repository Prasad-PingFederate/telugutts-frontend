# Telugu Text-to-Speech (TTS)

A simple and fast Telugu Text-to-Speech web application that converts Telugu text into natural MP3 audio.  
This project uses:

- **Frontend:** HTML, CSS, JavaScript  
- **Backend API:** Vercel Serverless Function  
- **Speech Engine:** RunPod Serverless (Python + gTTS)
